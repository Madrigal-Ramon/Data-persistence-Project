# Simple Restaurant Order System

## Problem
Manage restaurant orders with pending and completed lists that save automatically.

## Features
- **ArrayList**: Stores pending orders (grows automatically)
- **Array**: Stores completed orders (fixed size)
- **File Persistence**: Saves/loads orders from files
- **Error Handling**: Recursive input validation and try-catch blocks
- **Menu Interface**: Easy-to-use console menu

## How to Run
```bash
javac RestaurantOrderSystem.java
java RestaurantOrderSystem
